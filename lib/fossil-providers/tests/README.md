# Fossil Test Cases

This test suite contains a set of test cases for the Fossil provider.
Those are defined in the exact same way as the [RML test cases](https://rml.io/test-cases/).

| Test case  | Description                                                                                                                                        | Result |
|------------|----------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| RMLTC0000  | Tests if an empty table produces an empty RDF graph                                                                                                | OK     |
| RMLTC0001a | Tests: (1) one column mapping; (2) subject URI generation by using rr:tmplate; (3) one column to one property                                      | OK     |
| RMLTC0001b | Tests: (1) one column mapping; (2) generation of a BlankNode subject by using rr:termType; (3) one column to one property                          | OK     |
| RMLTC0002a | Tests: (1) two column mapping, no primary key; (2) subject URI generated by the concatenation of two column values; (3) one column to one property | OK     |
| RMLTC0002b | Tests: (1) two column mapping, no primary key; (2) generation of a BlankNode subject by using rr:template; (3) one column to one property          | OK     |
| RMLTC0002c | Tests the presence of an undefined SQL identifier / logical reference                                                                              | ERROR  |
